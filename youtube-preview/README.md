# Miniads YouTube Preview

Prévisualisation locale d'une miniature + d'un titre sur les tuiles du feed YouTube.

## Installation
1. Ouvre `chrome://extensions`
2. Active **Mode développeur**
3. **Charger l’extension non empaquetée** → sélectionne ce dossier
4. Va sur `https://www.youtube.com/` et actualise

## Utilisation
- Sur chaque tuile vidéo, clique **“Ajouter votre miniature”**
- Dans le modal :
  - Upload **ou** colle l’URL de ta miniature
  - Saisis un **titre**
  - Clique **Appliquer** (cible actuelle) **ou** **Appliquer à toutes**
- **Réinitialiser** = retire l’overlay + restaure le titre original

## Options (popup)
- Icône de l’extension → **Activer sur YouTube** (ON/OFF)

## Notes
- Tout est **local** (aucune modification côté YouTube).
- Le **hover preview** YouTube est bloqué pour laisser visible ta miniature.
