FAQ

Comment installer l’extension ?

Depuis Chrome, clique sur « Installer l’extension » ou charge le dossier en mode développeur (instructions ci-dessous).

Quelles permissions sont nécessaires ?

Uniquement l’accès à youtube.com, le stockage local pour enregistrer tes préférences et la gestion des onglets pour la popup.

Est-ce compatible avec le nouveau design YouTube ?

Oui, nous ciblons les composants Polymer (home, search, channel, watch “Up next”). Les styles sont isolés via Shadow DOM.

Installation développeur

  1. Ouvre chrome://extensions.
  2. Active le Mode développeur.
  3. Clique sur Charger l’extension non empaquetée.
  4. Sélectionne le dossier /yt-thumbnail-previewer-pro.
  5. Recharge YouTube puis clique sur l’icône « Visualisateur YouTube ».

Tu peux ensuite utiliser le menu de l’icône ou le raccourci Ctrl+Shift+Y pour ouvrir la popup avec YouTube.
