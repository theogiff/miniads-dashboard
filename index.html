Comment l’installer ?

Suivez ces étapes (2 minutes) et testez votre prochaine miniature.

  1. Ouvrez le Chrome Web Store via le bouton “Installer l’extension”.
  2. Ajoutez l’extension Miniads et épinglez-la dans Chrome.
  3. Dans “Mes miniatures”, cliquez sur “Voir sur le feed YouTube” pour précharger votre visuel.
